How I Got Into Coding

My first experience coding was in an online game named Graal. It had a level editor and you could script your own custom items people could use. I'd spend hours on end making things for fun not realizing just how much the NPC/Item scripting process had in common with using a standard programming language.

Jump to 2020, the pandemic stomps on my previous field of work. Left with spare time I decided to finally learn to code and signed up for TrueCoders coding bootcamp. I wish I had learned this stuff years ago. It's so satisfying to do and learn that I find myself losing track of time.
